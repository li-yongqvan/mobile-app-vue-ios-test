## 5. 网络连通性特攻解决方案
针对复杂的网络环境（如代理拦截、CORS 跨域、非标准端口 8000），本项目采用了 **Vite 反向代理 (Reverse Proxy)** 技术：

```javascript
// vite.config.js 配置选段
server: {
  proxy: {
    '/api': {
      target: 'http://114.132.68.81:8000',
      changeOrigin: true
    }
  }
}
```
**技术优势**：前端请求本地 `localhost:5173/api`，由开发服务器内部转发。通过物理隔离的方式彻底规避了用户代理软件拦截外部非标准端口请求的问题。
